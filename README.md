## Run Locally

Clone the project

```bash
  git clone https://github.com/sriharsh1104/solana_program_deployment.git
```

Go to the project directory

```bash
  cd program_deploy_process/programs/program_deploy_process
```

Install dependencies

```bash
  anchor build
```

Start the server

```bash
  anchor deploy
```
